# Autonomous Vehicle System using Deep Learning

## Features
- YOLOv8 Vehicle Detection
- DeepSORT Vehicle Tracking
- Lane Detection
- Collision Warning System

## Technologies Used
- Python
- OpenCV
- YOLOv8
- DeepSORT
- NumPy

## Project Structure

```bash
AutonomousVehicleProject/
│
├── datasets/
├── outputs/
├── models/
│
├── final_autonomous_system.py
├── requirements.txt
└── README.md
```

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python final_autonomous_system.py
```

## Output

The output video is saved inside:

```bash
outputs/final_output.avi
```
